# Windows のキーボードショートカット
tag: Windows 初心者向け 新人教育

キーボードショートカットとは、コンピュータを操作するためにキーボードを使って、特定の操作やコマンドを素早く実行する方法のことです。これにより、マウスを使わずに多くの操作を効率的に行うことができます。ここでは、18 歳の新入社員向けに Windows のキーボードショートカットを説明します。

## キーボードショートカットを使う目的
キーボードショートカットを使う主な目的を5つ挙げます。

### 作業が早くなる
キーボードショートカットを使うと、パソコンの操作がとても速くなります。たとえば、文章をコピーする時、マウスを使って「右クリック」して「コピー」を選ぶよりも、キーボードで「Ctrl」キーと「C」キーを同時に押す方がずっと速いです。短い時間でたくさんの作業を終わらせることができます。

### 手が楽になる
マウスを使ってあちこちクリックするよりも、キーボードだけで操作する方が手が疲れにくいです。キーボードショートカットを使うと、手や腕にかかる負担が減ります。

### 集中力が続く
キーボードショートカットを使うと、手をキーボードから離さずに作業ができるので、集中力が途切れにくくなります。たとえば、何かを書いている時に、わざわざマウスに手を伸ばす必要がなくなるので、集中して作業を続けられます。

### クールに見える
キーボードショートカットをスムーズに使いこなせると、周りから見てとてもかっこよく見えます。友達やクラスメートから「パソコンの達人みたい！」と言われるかもしれません。

### 障害を持つ人の助けになる
視覚に障害がある人や、マウスを使うのが難しい人にとって、キーボードショートカットはとても便利です。これを使うことで、より多くの人がパソコンを便利に使えるようになります。

## キーボードショートカットの基本操作

キーボードショートカットは、通常、1つのキーまたは複数のキーを同時に押すことで機能します。たとえば、「Ctrl」と「C」を同時に押すとコピー、「Ctrl」と「V」を同時に押すと貼り付けができます。

## テキスト編集のキーボードショートカット

テキスト編集でよく使われる10個のショートカットキーを紹介します。

|押すキー|目的|
|---|---|
|**`Ctrl`** + **`C`**|コピー|
|**`Ctrl`** + **`X`**|切り取り|
|**`Ctrl`** + **`V`**|貼り付け|
|**`Ctrl`** + **`Z`**|元に戻す|
|**`Ctrl`** + **`Y`**|「元に戻す」の取り消し|
|**`Ctrl`** + **`S`**|上書き保存|
|**`Ctrl`** + **`N`**|新規作成|
|**`Ctrl`** + **`P`**|印刷|
|**`Ctrl`** + **`A`**|全て選択|
|**`Ctrl`** + **`F`**|検索|

## デスクトップ操作のキーボードショートカット

Windows のデスクトップ操作でよく使われる16個のショートカットキーを紹介します。

|押すキー|目的|
|---|---|
|**`Tab`**|入力欄の移動|
|**`Shift`** + **`Tab`**|入力欄の逆移動|
|**`Ctrl`** + **`D`**|ゴミ箱に移動|
|**`Ctrl`** + **`R`**|作業中のウィンドウを最新の情報に更新|
|**`Alt`** + **`←`**|前のページに戻る|
|**`Alt`** + **`→`**|次のページに進む|
|**`Alt`** + **`Tab`**|開いているウィンドウの切り替え|
|**`Alt`** + **`F4`**|使用中の項目を閉じる、または作業中のプログラムを終了|
|**`Alt`** + **`Enter`**|プロパティの表示|
|**`Ctrl`** + **`Shift`** + **`N`**|フォルダの新規作成|
|**`Ctrl`** + **`Alt`** + **`Del`**|ソフトの強制終了|
|**`Ctrl`** + **`Shift`** + **`Esc`**|タスクマネージャーの起動|
|**`Ctrl`** + **`スクロールホイール`**|アプリの項目を拡大縮小する|
|**`Shift`** + **`F10`**|右クリックメニューの呼び出し|
|**`Shift`** + **`Del`**|ゴミ箱に移動せずに削除|
|**`Ctrl`** + **`左クリック`**|複数のファイルを個別に選択|
|**`Shift`** + **`左クリック`**|ファイルをまとめて選択|

## Windows キーのショートカット

Windows キーを使った13個のショートカットキーを紹介します。

|押すキー|目的|
|---|---|
|**`田`**|スタート メニューを開くまたは閉じる|
|**`田`** + **`D`**|デスクトップの表示・非表示|
|**`田`** + **`L`**|ユーザーの切り替え|
|**`田`** + **`M`**|全てのウィンドウを最小化|
|**`田`** + **`←`**|ウィンドウをデスクトップの左に寄せる|
|**`田`** + **`→`**|ウィンドウをデスクトップの右に寄せる|
|**`田`** + **`↑`**|ウィンドウサイズを拡大|
|**`田`** + **`↓`**|ウィンドウサイズを縮小|
|**`田`** + **`E`**|エクスプローラーを開く|
|**`田`** + **`P`**|ディスプレイの選択画面を開く|
|**`田`** + **`R`**|ファイル名を指定して実行する|
|**`田`** + **`M`**|全てのウィンドウを最小化|
|**`田`** + **`Tab`**|デスクトップの切り替え|

## ファンクションキーのショートカット

ファンクションキーを使った12個のショートカットキーを紹介します。

|押すキー|目的|
|---|---|
|**`F1`**|ヘルプを開く|
|**`F2`**|ファイル名・フォルダ名の変更|
|**`F3`**|ファイルやフォルダの検索|
|**`F4`**|Internet のアドレスバーに履歴を表示|
|**`F5`**|作業中のウィンドウをさいしんの情報に行為sン|
|**`F6`**|ひらがなに変換|
|**`F7`**|全角カタカナに変換|
|**`F8`**|半角カタカナに変換|
|**`F9`**|全角英数に変換|
|**`F10`**|半角英数に変換|
|**`F11`**|ウィンドウを全画面で表示|
|**`F12`**|名前をつけて保存|

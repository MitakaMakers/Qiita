# Excel のショートカットキーの紹介
tag: Excel 環境構築 初心者向け 新人教育

基本的に、パソコンの操作はキーボードやマウスを使って行います。
複数のメニュー階層を辿る操作でも、キーボードを使うと一発で終えることができる場合があります。そういった機能の事を「ショートカットキー」と呼びます。ショートカットとは「近道」の意味です。
ここでは 18 歳の新入社員向けに Excel のショートカットキーを紹介します。

## 一般的なキーボード ショートカット

|押すキー|目的|
|---|---|
|**`Ctrl`** + **1**|「セルの書式設定」ダイアログボックスの表示|
|**`Ctrl`** + **`S`**|ファイルを上書き保存|
|**`Ctrl`** + **`Z`**|操作を元に戻す|
|**`Ctrl`** + **`C`**|コピー|
|**`Ctrl`** + **`X`**|切り取り|
|**`Ctrl`** + **`V`**|貼り付け|
|**`Ctrl`** + **`A`**|全て選択|
|**`Ctrl`** + **`Y`**|直前の操作を繰り返す|
|**`Ctrl`** + **`F`**|文字列を検索|
|**`Ctrl`** + **`H`**|複数個所の文字列を置換|
|**`Ctrl`** + **`P`**|印刷プレビュー画面を開く
|**`Ctrl`** + **`B`**|太字の設定 / 解除|
|**`Ctrl`** + **`I`**|斜体の設定 / 解除|
|**`Ctrl`** + **`U`**|下線の設定 / 解除|
|**`Ctrl`** + **`D`**|上のセルのコピー＆貼り付け|
|**`Ctrl`** + **`R`**|左のセルのコピー＆貼り付け|
|**`Ctrl`** + **`G`**|ジャンプ ダイアログボックスの表示|
|**`Ctrl`** + **`スペース`**|列を全選択|
|**`Shift`** + **`スペース`**|行を全選択|
|**`Ctrl`** + **`－`**|(セル、行や列全体を選択した状態で)セル、行、列の削除|
|**`Ctrl`** + **`Shift`**  + **`＋`**|セル、行、列の挿入|
|**`Ctrl`** + **`Shift`** + **`:`**|表全体を選択する
|**`Alt`** + **`Enter`**|セル内で改行（セル選択時）|
|**`F2`**|セル内を編集|
|**`F9`**|開いているブックの再計算|

## Excel 上級者向けのショートカット

|押すキー|目的|
|---|---|
|**`Ctrl`** + **`N`**|ブックの新規作成|
|**`Ctrl`** + **`;`**|現在の日付を入力|
|**`Alt`** + **`Shift`** + **`=`** | SUM 関数入力|
|**`Ctrl`** + **`Enter`**|複数のセルへ一括入力|
|**`Ctrl`** + **`矢印キー`**|データの最後まで移動|
|**`Ctrl`** + **`Shift`** + **`矢印キー`**|データの最後まで一括選択|
|**`Ctrl`** + **`PageUp`**|左のシートに移動|
|**`Ctrl`** + **`PageDown`**|右のシートに移動|
|**`Ctrl`** + **`Shift`** + **`＋`**|(セル、行や列全体を選択した状態で)セル、行、列の挿入|
|**`Ctrl`** + **`L`**|フィルター|
|**`Ctrl`** + **`T`**|「テーブルの作成」 ダイアログボックスの表示|
|**`Ctrl`** + **`K`**|「ハイパーリンクの挿入」ダイアログボックスの表示|
|**`Ctrl`** + **`Shift`** + **`～`**|「標準」書式を設定|
|**`Ctrl`** + **`Shift`** + **`3`**|「日付」書式を設定|
|**`Ctrl`** + **`Shift`** + **`4`**|「通貨」書式を設定|
|**`Ctrl`** + **`Shift`** + **`5`**|「パーセンテージ」書式を設定|
|**`Shift`** + **`F2`**|コメントの挿入 / 編集|
|**`Shift`** + **`F9`**|現在編集中のワークシートを再計算|
|**`Shift`** + **`F11`**|新しいワークシートを挿入|
|**`Alt`** + **`；`**|選択範囲の可視セルを選択|
|**`Alt`** + **`F4`**|Excel を終了する|
|**`Alt`** + **`Tab`**|ウィンドウの切替|
|**`F11`**|新規グラフシートの挿入|
|**`F12`**|名前を付けて保存|

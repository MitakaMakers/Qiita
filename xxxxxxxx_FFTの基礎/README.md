# FFT解析の紹介

## FFT解析の機械・電気工学での使用例

+ 【用途1】室内音響、スピーカー設計、環境騒音解析(- 20KHz)
+ 【用途2】機械的衝撃応答試験、落下試験(- 20KHz)
　　　振動測定
+ 【用途3】機構や建築物の構造動的解析、ねじり解析(- 20KHz)

+ 【用途5】燃焼分析(- 20KHz)
+ 【用途6】交流電流の高調波、有効電力の(- 50KHz)
　　交流電流の高調波
　　交流電流の有効電力・無効電力・皮相電力
+ 【用途4】回転機械の状態の予測モニタリング(- 500KHz)
　　耐久性、疲労解析、ベアリング故障検出
+ 【用途7】制御回路の周波数特性(- 1MKHz)
　　ボード線図

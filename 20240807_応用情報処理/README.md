応用情報技術者試験

応用情報技術者試験（以下、応用情報と呼びます）は、基本情報技術者試験の上位に位置する試験で、より高度なIT知識とスキルを持っていることを証明する国家試験です。ここでは 22 歳の新入社員向けに応用情報の概要を説明します。

# 応用情報を受検する目的
応用情報を受検すると、次の知識と技能を得ることができます。

### システム設計
コンピュータシステムの設計方法を学びます。どのようにシステムを作るか、効率的に動かすかを考えます。

### プロジェクト管理
プロジェクトを計画し、管理する方法を学びます。スケジュール管理やチームの運営方法なども含まれます。

### IT戦略
企業や組織のためにITをどう活用するかを学びます。経営戦略とITを結びつける力を養います。

# 応用情報の受験方法
応用情報は秋季と春季の年2回実施されています。申し込み後、指定された試験日に試験会場にてコンピュータを用いる方式によって実施されます。

具体的な申し込み方法は、試験実施団体である情報処理推進機構（IPA）のホームページよりご確認ください。
https://www.ipa.go.jp/shiken/mousikomi/cbt_sg_fe.html

# 応用情報の出題範囲
応用情報処理試験は出題範囲が広いことで知られています。おおまかには知識を問う「午前問題」と技能を問う「午後問題」に分けられます。


|試験区分|科目名|試験時間|出題数|出題形式|合格点|配点|
|---|---|---|---|---|---|---|
|応用情報技術者試験|午前|150分|80問|四肢択一|60点|100点|
|応用情報技術者試験|午後|150分|11問|記述式|60点|100点|

# 午前の出題範囲
午前の試験はテクノロジ、マネジメント、ストラテジ分野の応用的な知識を問う問題が 80 問出題されます。
